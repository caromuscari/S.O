Estructura de mensajes (Paquetes estandar de 1024 bytes)
H + YY + LLLL + XXXXXXXXXXXXXXXXXXXXXXXX

H: Identificador del emisor 		(1 bytes)
Y: Identificador del mensaje 		(2 bytes)
L: Longitud del mensaje a enviar (4 bytes)
X: Texto del mensaje a enviar 	(1017 bytes) - en caso de vacio debe ser caracter nulo
							TOTAL =	1024 bytes

------- DICCIONARIO DE MENSAJES -------
KERNEL - K
K00: Mensaje para pedir identificarse al recien conectado
K01: Mensaje para pedir datos (vale para cualquiera)
K02: Identificarse a Memoria
K03: Identificarse a FileSystem

CPU - P

MEMORIA - M
M-1: Respuesta de la memoria a cualquier Cliente (K o P), indica que NO PUEDE responder el pedido.
M00: Respuesta a la solicitud de datos (Kernel) de tamaño de página.

CONSOLA - C

FILESYSTEM - F
